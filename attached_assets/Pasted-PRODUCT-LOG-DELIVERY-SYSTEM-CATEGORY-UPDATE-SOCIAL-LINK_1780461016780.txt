Automatic Log Delivery

When a user purchases a product:

1. Verify wallet balance.
2. Deduct purchase amount.
3. Create order record.
4. Automatically assign one available log from that product.
5. Mark assigned log as sold.
6. Prevent the same log from being sold twice.
7. Save assigned log to the user's account.
8. Record transaction history.
9. Update analytics.
